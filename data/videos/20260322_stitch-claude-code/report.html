Overview
📌核心概述

Stitch 2.0 是Google发布的全新AI设计工具，集成Gemini 3.1模型， 专门解决 Claude Code 等AI编码工具在前端设计方面的短板。 通过"灵感 → Stitch → Claude Code"的工作流，用户可快速生成高质量、看起来不像AI垃圾的网站。

💡 关键洞察

Stitch发布后，Figma股价应声下跌近8%，因为它为非专业设计师提供了免费且强大的原型设计能力。

Workflow
🔄推荐工作流程
1

寻找灵感

Dribbble / Godly / Pinterest找设计风格

2

截图 + 描述

粘贴截图并用自然语言描述需求

3

AI生成设计

自动创建设计系统和多版本原型

4

导出到Code

复制代码到Claude Code完善后端

Features
⚡Stitch 2.0 核心功能

🎨 设计系统生成

自动从截图和提示词中提取设计规范

  • 主色/次色/中性色定义
  • 字体选择
  • 圆角规范
  • 组件样式

🔄 快速迭代

一键生成多个变体版本

  • 布局变体
  • 配色变体
  • 图片变体
  • 自定义创意范围

✏️ 可视化编辑

点击即可编辑单个组件或整体布局

  • 组件级编辑
  • 布局级编辑
  • 实时预览
  • 全屏模式

🎥 Live Mode

AI实时观察屏幕并对话交互

  • 语音控制设计修改
  • 动态背景添加
  • 鼠标跟随效果

📋 设计文档

自动生成完整设计文档

  • 设计概述
  • 创意方向
  • 色彩策略
  • 排版规范

📤 导出选项

多种导出方式

  • 复制代码到剪贴板
  • 发送到AI Studio
  • 导出到Figma
  • 发送到Jules
