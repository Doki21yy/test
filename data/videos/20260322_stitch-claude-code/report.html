2. 截图上传 → 在Stitch中粘贴截图，用简单语言描述需求

3. 选择变体 → 使用explore功能获取3种不同方案

5. 代码导出 → 一键复制代码到Claude Code

6. 后端完善 → Claude Code处理交互逻辑和后端连接

7. 部署上线 → GitHub + Vercel完成部署
